# Market Booth CRM

**[🔗 GitHub Repo](https://github.com/AaronC1992/market-booth-crm)**

---

A flea market booth management CRM built with Vite + React + TypeScript + Tailwind CSS. Features an interactive floor plan, booth status tracking, vendor management, and payment history.

## Features

- Visual flea market floor plan with color-coded booth status
- Booth detail panel (vendor info, status, pricing)
- Vendor and payment management pages
- Dark mode support
- Responsive layout with sidebar navigation

## Tech Stack

- [Vite](https://vitejs.dev) + [React 18](https://react.dev) + [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS v4](https://tailwindcss.com)
- [React Router DOM v6](https://reactrouter.com)
- [Lucide React](https://lucide.dev)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.
